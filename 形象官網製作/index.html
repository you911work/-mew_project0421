創業理念

可持續發展的未來

我們正在建立一個以太陽能提供能源、以電池提供動力、以電動車代步的環保世界。了解我們的產品、員工和供應鏈的最新資訊。

讓一切變得更有趣

我們的車輛是世界上最安全的車輛。在考慮安全時，我們的目標是讓每輛 Tesla 讓你享受最大的駕駛樂趣。我們提供各種功能，讓你在車內更愉快，包括遊戲、電影、各種隱藏功能等等。透過無線軟件更新，我們定期推出功能，只需一按即可更新。

製造零件的機器

為了使人類棄用化石燃料，我們需要擴展至極端規模。我們在德州設立總部，並於三大洲營運六間工廠。我們的團隊擁有超過 100,000 名員工，自行設計、建造、銷售及維修我們的產品。

團隊信念

我們採用「第一原理」，解決了世界上最大的一些難題。如果你有出色的成就，歡迎加入我們，共同應對新一代工程、製造業和營運方面的挑戰。

價格說明

Tesla Model 3
NT$175.5萬起

Model 3 的內飾與眾不同。您將能使用智慧型手機取代車鑰匙，並透過 15 吋的中央觸控式螢幕，查看及調整各項控制功能。全景玻璃車頂由前方延伸至最後，為每個座椅打造寬敞空間。

Tesla Model S
NT$175.5萬起

Model S Plaid 擁有所有量產車輛中最快的加速。Model S 動力傳送系統均搭配最新的電池架構，能夠連續穩定地完成加速。

Tesla Model Y
NT$212萬起

Model Y 是完全使用電力驅動的車輛，因此您再也無須開到加油站加油。如果您晚上在家充電，次日清晨即可滿電出發。

Tesla Model X
NT$310.6萬起

Model X Plaid 具有其他 SUV 無可匹敵的動力與加速，是史上性能最強大的 SUV 之一。最新的電池架構讓 Long Range 和 Plaid 配置能夠在性能沒有下降的情況下完成連續賽道體驗。
